### Task 1: Check if a Number is Even or Odd
This program takes a number as input from the user and checks whether the number is even or odd.

**Logic:**
- The program uses the modulus operator `%`.
- If a number is divisible by 2 (`num % 2 == 0`), it is an **even number**.
- Otherwise, it is an **odd number**.

**Example:**
Input: `4`  
Output: `4 is an even number`

Input: `7`  
Output: `7 is an odd number`

### Task 2: Sum of Integers from 1 to 50 Using a Loop
This program calculates the sum of numbers from **1 to 50** using a `for` loop.

**Logic:**
- A variable `sum` is initialized to `0`.
- A `for` loop iterates through numbers from `1` to `50`.
- Each number is added to the `sum` variable.
- Finally, the total sum is printed.

**Output:**
The sum of numbers from 1 to 50 is: `1275`
